\vskip.15in
\noindent\textbf{Course objectives:}  The idea behind this course is to gain exposure and learn the fundamentals of machine learning through weekly discussions and exercises. Secondary goal is to republish the code that is used throughout the text and excersises in the form of jupyter notebooks in python. The motivation is for that is twofold. 1. Current implemntations are in Matlab, which some people lack access to. 2. More learning will happen doing things from scratch.
